Блок навигации

  • Ширина 15em
  • Должен присутсвовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступ снизу равным .2em, при помощи свойства magin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступ снизу равным .2em, при помощи свойства magin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внуть контентом.
